## 📖 About
**LearnBeam** is an AI-powered educational web application designed to act as an interactive 24/7 personal tutor. It transforms static study materials into dynamic conversations, helping students understand complex topics, summarize lectures, and prepare for exams efficiently.

## ✨ Key Features
* **🤖 AI Tutor:** Ask questions and get instant, context-aware explanations.
* **📝 Document Summarization:** Upload PDFs or notes to get concise summaries and key takeaways.
* **⚡ Smart Q&A:** Context-specific answers based on your uploaded course materials.
* **🔒 Student-Centric:** Designed for ease of use and focused on academic integrity.

## 🚀 Getting Started

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/HoloHive/LearnBeam.git](https://github.com/HoloHive/LearnBeam.git)
    cd LearnBeam
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    pip install -r requirements.txt
    ```

3.  **Run the application**
    ```bash
    npm run dev
    ```
